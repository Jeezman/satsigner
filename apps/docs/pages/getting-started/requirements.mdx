# SatSigner Requirements

To start with `satsigner` you must have the following requirements installed on your machine:

- [node.js](https://nodejs.org)
- [yarn](https://yarnpkg.com/getting-started/install)

We recommend using Visual Studio Code for the code editor, if you plan to contribute:

- [Visual Studio Code](https://code.visualstudio.com/)

Next, Follow the expo documentation [here](https://docs.expo.dev/guides/local-app-development) or read the instructions below.

## Android

Get [Android Studio](https://developer.android.com/studio)
